bbHierarchicalGrid is hierarchical data grid made for WPF and WinForms.

Getting Started

bbHierarchicalGrid is available via NuGet:

Copy
Package-Install bbHierarchicalGrid

  • WPF

    bbHierarchicalGrid has three themes: Default, Light and Dark. Add a resource dictionary to your App.xaml. Inspect the following example:

    XAML
    Copy
       <Application x:Class="bbHierarchicalGrid.WPF.Test.App"
             xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
             xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
             xmlns:local="clr-namespace:bbHierarchicalGrid.WPF.Test"
             StartupUri="MainWindow.xaml">
    <Application.Resources>
        <ResourceDictionary>
            <ResourceDictionary.MergedDictionaries>
                <ResourceDictionary Source="/bbHierarchicalGrid;component/Styles/Light.xaml" />
            </ResourceDictionary.MergedDictionaries>
        </ResourceDictionary>
    </Application.Resources>
</Application>
    1. You need to instantiate the bbViewModel class with your license key.
    2. Initialize the view model.
    3. Set the bbHierarchicalDataGrid's datacontext to view model.

    The code sample below demonstrates that:

    C#
    Copy
    using bbHierarchicalGrid.ViewModels;
using System.Windows;
using WPF.Models;
namespace WPF.Views
{
    /// <summary>
    /// Interaction logic for MainWindow.xaml
    /// </summary>
    public partial class MainWindow : Window
    {
        public MainWindow()
        {
            // create a new instance of view model
            // we are setting this instance to static property called ViewModel
            ViewModels.MainWindowViewModel.ViewModel = new bbViewModel<Item>(Properties.Settings.Default.LicenseKey);
            // Initialize view model (create columns)
            ViewModels.MainWindowViewModel.ViewModel.Initialize();
            // Initialize components of the main window
            InitializeComponent();
            // set data context of the bbHierarchicalGrid usercontrol 
            this.bbGrid.DataContext = ViewModels.MainWindowViewModel.ViewModel;
        } 
    }
}
  • WinForms

    bbHierarchicalGrid styling requires you create a WPF application inside your WinForms application and load the correct theme. See below.

    bbHierarchicalGrid is usable in WinForms applications via WPF interoperability. The sample code below demonstrates how to setup bbHierarchicalGrid in the code-behind of a WinForms window:

    C#
    Copy
     bbViewModel<Country> viewModel = default(bbViewModel<Country>);
 public WinForms()
 {

     // create a new WPF application 
     var wpfApp = new System.Windows.Application();
     var theme = new ResourceDictionary();

     /*
      * Themes are:
      * Default
      * VisualStudio.Blue
      * VisualStudio.Light
      */
     theme.Source = new Uri("pack://application:,,,/bbHierarchicalGrid;Component/Styles/Dark.xaml");
     wpfApp.Resources.MergedDictionaries.Add(theme);

     // set display language in the constructor of the form
     viewModel = new bbViewModel<Country>(Properties.Settings.Default.LicenseKey);
     // create columns
     viewModel.Initialize();
     InitializeComponent();
 }

 private void WinForms_Load(object sender, EventArgs e)
 {

     // create a new instance of bbHierarchicalGrid usercontrol
     var bbHierarchicalDataGridControl = new bbHierarchicalGrid.bbHierarchicalDataGrid();
     // Drag and drop an ElementHost control on the designer from the toolbox to create instance
     this.elementHost1.Child = bbHierarchicalDataGridControl;
     // hook viewmodel into the view
     bbHierarchicalDataGridControl.DataContext = viewModel;
     // todo: add items
}
