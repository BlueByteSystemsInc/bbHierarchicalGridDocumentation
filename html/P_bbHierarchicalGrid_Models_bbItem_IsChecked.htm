Remarks
  • This column has a default order of -1. To set column order, use bbColumnOrderAttribute.
  • This column also has an empty column header. To set column header, use DescriptionAttribute.
  • GetAllChildrenT(T) is a utility method that returns all children. Use this method to propagate state (IsChecked) if overridden. The code sample below shows an overridden implementation of IsChecked:
    C#
    Copy
    private bool isChecked = false;
/// <summary>
/// Gets or sets the item's IsChecked.
/// </summary>
    [Description("This is a the header.")]
    [bbColumnOrder(1)]
    public override bool IsChecked
    {
        get { return isChecked; }
        set
        {
            var children = this.GetAllChildren<ConcreteItem>();
            if (children != null)
                if (children.Length > 0)
                    Array.ForEach(children, x => x.IsChecked = value);
            SetProperty(ref isChecked, value);
        }
    }
