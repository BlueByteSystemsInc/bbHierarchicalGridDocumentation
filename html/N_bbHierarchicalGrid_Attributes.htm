bbHierarchicalGrid.Attributes Namespace

Attributes namespace for the bbHierarchicalGrid library.
Classes
  ClassDescription
bbColumnOrderAttribute
Decorates a property. This attribute sets the column order from left to right.
bbEditablePropertyAttribute
Binds the property, this attribute decorates, to a textbox.
bbIgnorePropertyAttribute
Decorates a property. This attribute hides the column a property is bound to. Similar effect of BrowsableAttribute.
bbNumericUpPropertyAttribute
Decorates numeric properties. Rends the property decorated by this attribute in NumericUpDown control.
bbSortableAttribute
Decorates a property. Rends a column, a property is bound to, sortable.
bbUseDateTimePickerAttribute
Decorates DateTime properties. Rends the property decorated by this attribute in DateTimePicker (Xceed.Wpf.Toolkit.DateTimePicker).
